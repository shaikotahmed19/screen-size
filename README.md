# Screen Size Catcher

**Live Demo:** https://shaikotahmed19.github.io/screen-size/

## Overview

Screen Size Catcher is an elegant, real-time screen resolution and browser viewport dimension tracker. Instantly inspect, track, and copy your device and browser size metrics with a beautiful, responsive interface.

## Features

- **Real-time Screen Resolution Tracking** - Display your total screen dimensions with device pixel ratio, color depth, and orientation
- **Browser Viewport Monitoring** - Track active window size with live updates during resizing
- **Aspect Ratio Detection** - Automatically identifies common aspect ratios (16:9, 4:3, 21:9, etc.) or shows custom simplified ratios
- **Visual Preview** - Interactive ratio box that scales proportionally to your dimensions
- **One-click Copy** - Copy dimensions to clipboard with visual feedback and toast notifications
- **Theme Switching** - Toggle between Default, Dark, and Light themes with persistent storage
- **Responsive Design** - Works seamlessly on all screen sizes and devices
- **Touch Detection** - Identifies if your device supports touch interface

## What It Shows

### Total Screen Size
- Screen width and height in pixels
- Device Pixel Ratio (DPR)
- Color depth
- Device orientation (Portrait/Landscape)
- Aspect ratio with visual preview

### Browser Window Size
- Viewport width and height
- Outer window dimensions
- Touch interface support
- Real-time updates during window resize
- Aspect ratio with visual preview

## Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Modern styling with CSS variables for theming
- **JavaScript (Vanilla)** - No dependencies, pure JavaScript
- **Google Fonts** - JetBrains Mono and Outfit fonts
- **Responsive Design** - Mobile-first approach

## Theme Support

Switch between three themes:
- **Default** - Dark theme optimized for coding
- **Dark** - Pure dark mode
- **Light** - Light mode for daytime use

Theme preference is saved in localStorage for persistence across sessions.

## Usage

1. Open the application in your browser
2. View your screen and viewport metrics in real-time
3. Resize your browser window to see live updates
4. Click "Copy" buttons to copy dimensions to your clipboard
5. Toggle themes using the dropdown menu in the header

## Installation & Setup

Simply clone or download the files and open `index.html` in your browser. No build process or dependencies required.
